🌌 과정별 소개
LEVEL 1
첫걸음
  • 별자리를 익히며 우주를 마주하는 과정
  • 다양한 교보재와 흥미로운 놀이 활동
LEVEL 2
체험
  • 다채로운 주제로 우주를 알아가는 과정
  • 재미있는 만들기와 조별 활동 진행
LEVEL 3
탐구
  • 호기심을 자극하여 우주를 찾아보는 과정
  • 계절마다 밝은 별, 행성 위주로 망원경 실습
LEVEL 4
테마
  • 깊이 있는 주제로 우주를 이해하는 과정
  • 전국 어린이천문대 '천체관측대회' 참가 자격
  • 수업 시간 9시로 변경됨 (망원경 실습 중심)
✏ 메모 — 레벨 흐름
  • 6~8월: 별꿈이 + 2시간 수업 - 만들기 / 관측
  • 9월~: 차기 정규과정 시작 (관측시간 → 30분+α)
📆 수업 신청 및 일정 선택
팀 구성
  • 적정 10명 ~ 최대 10명
  • 직접 팀 구성 or 모집 중인 팀에 합류
  • 팀 구성 어려우면 네이버카페 "천문대 함께해요" 게시판 활용
  • 월 1회, 연 12회 수업 (과정별)
  • 탐구교실로 시작 시 15회 수업
일정 선택 & OT
  • 화~토 중 선택
  • 7시부: 19:00 ~ 20:50
  • 9시부: 21:10 ~ 23:00
  • 팀 구성 완료 후, 담당 선생님과 OT 진행
  • 시작하는 학년에 따라 LEVEL 결정
✏ 메모
  • 9시부 → 늦게 끝남
  • 7시부 단점: 관측 시간이 짧음
  • 7시부 단점: 우리 반만 관측하는 것이 아닌 다른 반과 동시 관측
🌠 수업 진행
정규 수업
  • 매월 다른 내용의 프로그램(실습 포함)
  • 당일 불만한 대상을 관측 (월 1회, 2시간 수업)
  • 수업 후 단톡방으로 아이들 사진&선생님 후기 공유
날씨 관련
  • 비/눈/미세먼지 등 기상 관계없이 수업 진행
  • 야외 관측 불가 시 → 실내 관측 등 대체 프로그램
  • 법정공휴일과 수업날짜 겹치면 일시적 날짜 변경 진행
🔄 보강 수업
수업 보강
  • 정규 수업에 결석한 경우 참여 가능
  • 현장보강 또는 온라인 보강 중 택1 (중복 불가)
  • 다음 달 토요일 오후 3:30~5:00에 보강 진행
  • 탐구 기초과정의 경우 온라인 보강만 진행
  • 교육 보강 일정 변동 시 미리 공지
✏ 메모 — 토요일 보강 시간표
  • 1주차 토요일: LEVEL 1 (첫걸음과정) / LEVEL 2 (체험과정)
  • 3주차 토요일: LEVEL 3 (탐구과정) / LEVEL 4 (테마과정)
  • 한시간: 3:50 ~ 4:50
  • 별꿈이는 보강 없음
  • ex) 9월 결석 → 10월에 한번에 묶어서 보강 가능!
관측 보강
  • 2~3회 연속으로 별 관측 불가 시 관측 보강 진행
  • 정규 수업과 별개로 기상 예보와 구름 영상 확인 후 당일 연락(번개)으로 진행
✏ 메모 — 관측 보강 예시
  • 당일에 단톡 / 과반수 이상 동의 시 진행
  • 7월 수업 → 비 → 당월 진행
  • 8월 수업 → 흐림 → 관측 보강 진행
  • 9월 수업 → 흐림
🏅 LEVEL UP / 수료
  • 수료 전 대표 부모님을 통해 다음 과정 안내
  • 테마과정(LEVEL 4)은 망원경 실습 중심 → 9시부로 변경
  • 각 교육과정 수료 시 수료증 발급 (6회 이상 출석)
  • 과학의 재미와 교육효과 제대로 원하면 LEVEL 3~4 추천
